## Project Description
SCAMS is a web-based platform designed to help student clubs manage activities, registrations, attendance, and reporting in a centralized system.

## Tech Stack
- Frontend: React.js
- Backend: Node.js (Express)
- Database: MongoDB

## Current Progress (Week 2)
- GitHub repository created
- Project structure initialized (frontend & backend)
- Backend server setup started
- Database schema design in progress

## Features (Planned)
- User authentication (login/register)
- Role-based access (Member, Executive, Advisor)
- Activity management (CRUD)
- Event registration
- Attendance tracking
- Basic reporting
